# AUTO-DATASCIENTIST UI

Production-ready React frontend for the Auto-DataScientist platform.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- ✅ Cyberpunk glassmorphism theme
- ✅ Real-time workflow monitoring
- ✅ Live agent timeline updates
- ✅ Terminal log streaming
- ✅ Workflow history sidebar
- ✅ Generated code viewer
- ✅ Full FastAPI backend integration

## Backend Connection

The UI connects to FastAPI at `http://localhost:8000`.

Make sure your orchestrator is running:
```bash
python services/orchestrator/src/main.py
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Axios
- Google Material Icons
